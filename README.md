# My-Diet-Story

# 概要
このアプリは、ダイエットを行いたい女性が食事・運動等の記録を行えるWebアプリケーションです。また、ユーザー同士が掲示板に悩み等を投稿し、交流を行うことが出来ます。ダイエットが進むと、段階的にストーリーが解放されていく仕組みとなっています。

# サイトテーマ
「今度こそ挫折しないダイエットにする」

# テーマを選んだ理由
ダイエットを決意しても続けられない、挫折してしまう人は多いです（私含む）。ダイエットを楽しく成功させたい方向けに、長期的に続けられるように生活の中に溶け込ませたいと考えました。ユーザーは、ダイエットを継続する（投稿、目標達成、他ユーザーとの交流）とポイントがたまり、ストーリーが解放されます。さらに、目標達成のために他ユーザーとダイエット情報の交換もでき、モチベーションの維持もできます。

# きっかけ
自身が、人生最高体重となりダイエットが必要だと痛感していました。ダイエットを始めようとしても、甘えて運動をサボってしまったり、爆食をしてしまったりすることも多く、続けられる方法がないか模索していました。ダイエット＝辛いものと思っていたためです。私が女性向けのゲームが好きなので、好きなキャラクターがいればダイエットを頑張れるのと、一緒に頑張れる仲間がいればよりモチベーションの維持につながると感じたのがきっかけです。

# 問題点
ダイエットは孤独になりやすい  
結果が出にくい  
ダイエット＝辛いと思い込みモチベーションを保ちにくい 

# 解決案
ユーザー同士の交流・アドバイス  
続けるとストーリーが解放されるのでモチベーションアップ

# ターゲットユーザ
ダイエットをしたい10代以降の女性  
女性向けゲームが好きな人  
ダイエットを楽しく続けていきたい人

# 主な利用シーン
ダイエットの体重記録  
やったことのつぶやき  　
ユーザー同士の交流  

# 利用方法
Xか掲示板のように投稿ができる  
ストーリーが読める
